# ErgoFlow MVP

A local-first Chrome Extension for ergonomic health.

## Installation
1. Open Chrome and navigate to `chrome://extensions`.
2. Enable **Developer Mode** (top right).
3. Click **Load unpacked**.
4. Select the `ergoflow-mvp` folder generated by the setup script.

## Development
- Logic: `src/background.js`
- UI: `src/content.js` & `src/styles.css`
